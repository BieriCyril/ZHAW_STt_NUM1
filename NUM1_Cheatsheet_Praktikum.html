Partielle DGL – Wärmeleitung (W13)
Keine eigenen Funktionen – Algorithmen direkt im Code · W13
FTCS – Forward Time Centered Space
Explizites Schema für die Wärmeleitungsgleichung ∂T/∂t = a·∂²T/∂x². Stabil wenn s = τ·a/h² ≤ 0.5.

T[1, 1:-1] = s * (T[0, 2:] + T[0, :-2]) + (1 - 2*s) * T[0, 1:-1]

Mit Neumann-RB (Fluss): T[1, 0] = 2s·(T[0, 1] – T[0, 0] – h·l(t)) + T[0, 0]
Schlüsselvariablen
VariableBedeutung
sStabilitätszahl s = τ·a/h² (muss < 0.5 sein)
tauZeitschritt
hOrtsschritt
aTemperaturleitfähigkeit [m²/s]
Crank-Nicolson (implizit, W13)
Implizites Schema, Ordnung 2 in Zeit und Ort. Unbedingt stabil. Löst pro Zeitschritt ein tridiagonales LGS mit scipy.linalg.solve_banded.
Wichtige Parameter der Bandmatrix
VariableFormelBedeutung
alphaa / (2h²)Nebendiagonale
beta1/τ + a/h²Hauptdiagonale
gamma1/τ – a/h²Koeffizient rechte Seite
