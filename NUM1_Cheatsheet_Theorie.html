1 · Kondition eines linearen Gleichungssystems
Blatt15 A1
Begriffe & Schlagwörter (A–Z)
  • Eingabefehler / Störung
  • Fehlerfortpflanzung
  • Fehlervergrösserung
  • gut konditioniert
  • Konditionszahl
  • Matrixnorm
  • mindestens / höchstens Fehler
  • Näherungslösung
  • relativer Fehler
  • schlecht konditioniert
  • Sensitivität
  • Spaltensumme (1-Norm)
  • Verstärkungsfaktor
  • Zeilensumme (∞-Norm)
Formeln / Ausdrücke / Funktionen
  • A⁻¹ wird gegeben
  • A·x = b
  • b̃ ≈ b (gestörte rechte Seite)
  • cond(A) = ‖A‖ · ‖A⁻¹‖
  • ‖A‖₁ = max Spaltensumme |aᵢⱼ|
  • ‖A‖∞ = max Zeilensumme |aᵢⱼ|
  • ‖Δb‖/‖b‖ ≥ ‖Δx‖/‖x‖ / cond(A)
  • ‖Δx‖/‖x‖ ≤ cond(A) · ‖Δb‖/‖b‖
  • x̃ ≈ x (Näherungslösung)
  • 1 % Fehler x → wie gross Δb?
Worum geht es?

Wie stark verstärkt A einen Fehler in b? Grosse Konditionszahl = kleine Fehler in b → riesige Fehler in x.

Gegeben / Gesucht
Gegeben A, A⁻¹, rel. Fehler x̃
Gesucht cond(A) in einer Norm
Gesucht minimaler rel. Fehler b̃
Merksätze
cond ≈ 1 = gut · cond ≈ 10⁶ = 6 Stellen weg
Gleiche Norm für A und A⁻¹!
Beispiel: cond=50, Fehler x=1% → Fehler b ≥ 0.02%
Kochrezept
  1. 1-Norm: grösste Spaltensumme der Beträge
    ∞-Norm: grösste Zeilensumme der Beträge
  2. Konditionszahl:cond(A) = ‖A‖ · ‖A⁻¹‖
  3. Fehler umformen:‖Δb‖/‖b‖ ≥ ‖Δx‖/‖x‖ / cond(A)
2 · Lineare Gleichungssysteme — LR-Zerlegung
Praktikum W2/W3
Begriffe & Schlagwörter (A–Z)
  • Dreieckssystem lösen
  • Einheitsmatrix auf Diag. (L)
  • Gauss-Elimination
  • Indexvektor I
  • kompakte Speicherung
  • LR-Zerlegung
  • mehrere rechte Seiten
  • Multiplikator lᵢⱼ
  • Pivotisierung
  • relatives Spaltenmaximum
  • Rückwärtseinsetzen
  • sparse / dünnbesetzt
  • Tridiagonalmatrix
  • Vorwärtseinsetzen
  • Zeilenvertauschung
Formeln / Ausdrücke / Funktionen
  • A = L · R
  • A·x = b
  • backsub(R, y) → nur R
  • fbsub(L, R, b) → L und R
  • linsolve(A, b) → alles auto
  • LR(A) → L, R [ohne Pivot]
  • LR(A) → L, R, I [mit Pivot]
  • LR_tri(A) → kompakt
  • lᵢⱼ = aᵢⱼ / aⱼⱼ
  • pivot(A, I, k)
  • xᵢ = (yᵢ − Σⱼ﹥ᵢ Rᵢⱼxⱼ) / Rᵢᵢ
  • yᵢ = (bᵢ − Σⱼ﹤ᵢ Lᵢⱼyⱼ) / Lᵢᵢ
Gegeben / Gesucht
Gegeben Matrix A (n×n), Vektor b
Gesucht Lösung x: A·x = b
Gesucht L, R so dass A = L·R
Merksätze
Mehrere b? LR einmal, fbsub mehrfach.
backsub ≠ fbsub: backsub=nur R (QR). fbsub=L+R (LR).
Diag ≈ 0 ohne Pivot → instabil! linsolve macht Pivot automatisch.
Kochrezept
  1. LR ohne Pivot: L,R = LR(A) → x = fbsub(L,R,b)
  2. LR mit Pivot: L,R,I = LR(A) → b = b[I] → x = fbsub(L,R,b)
  3. Tridiagonal (FDM): LR = LR_tri(A)
  4. Vorwärts L·y=b:yᵢ = (bᵢ − Σⱼ﹤ᵢ Lᵢⱼ·yⱼ) / Lᵢᵢ
  5. Rückwärts R·x=y:xᵢ = (yᵢ − Σⱼ﹥ᵢ Rᵢⱼ·xⱼ) / Rᵢᵢ
3 · Cholesky-Zerlegung & Plattenkondensator
Praktikum W4/W5 · Blatt15 A4

Formeln / Ausdrücke / Funktionen
  • A = L · Lᵀ (mychol)
  • A = L · D · Lᵀ (von Hand)
  • AᵀA · p = Aᵀb
  • C = ε₀·εᵣ·A/d
  • CPlatte(epsr, A, d)
  • D·y = z → yᵢ = zᵢ/dᵢ
  • dᵢ = aᵢᵢ − Σₖ﹤ᵢ lᵢₖ²·dₖ
  • lᵢⱼ = (aᵢⱼ − Σₖ﹤ⱼ lᵢₖlⱼₖdₖ) / dⱼ
  • Lᵀ·p = y
  • L·z = b
  • mychol(A) → L
  • Δφ = 0 (Laplace)
Kochrezept — LDLᵀ von Hand (Blatt15 A4)
  1. Normalgleichung: AᵀA·p = Aᵀb
  2. LDLᵀ-Zerlegung:dᵢ = aᵢᵢ − Σₖ﹤ᵢ lᵢₖ²·dₖ lᵢⱼ = (aᵢⱼ − Σₖ﹤ⱼ lᵢₖ·lⱼₖ·dₖ) / dⱼ
  3. Vorwärts L·z = b
  4. Diagonal D·y = z: yᵢ = zᵢ/dᵢ
  5. Rückwärts Lᵀ·p = y
LLᵀ (mychol) hat √dᵢ auf Diag. LDLᵀ vermeidet Wurzeln → von Hand einfacher!
Nur möglich wenn alle dᵢ > 0 (positiv definit)!

Formeln / Ausdrücke / Funktionen
  • f(p) = 0 (überbestimmt)
  • fᵢ(p) = φ(xᵢ;p) − yᵢ
  • J[i,j] = ∂fᵢ/∂pⱼ = ∂φ/∂pⱼ
  • JᵀJ · Δp = −Jᵀ·f(p)
  • p_neu = p + Δp
  • tol = 10⁻⁹
  • y = A/x + B/x·e^(Cx²) (A6)

Kochrezept
  1. Residuenvektor: fᵢ(p) = φ(xᵢ;p) − yᵢ
  2. Jacobi J (n_Daten × n_Param): J[i,j] = ∂φ/∂pⱼ
  3. Schritt: JᵀJ·Δp = −Jᵀf → p += Δp
  4. Abbruch: ‖Δp‖ < tol
Schlechte Startwerte → kein Konvergenz → dann Levenberg-Marquardt (Abs. 7)
JᵀJ ist (n_Param × n_Param) — kleines System!

Formeln / Ausdrücke / Funktionen

Kochrezept

  1. g(θ) = forwardKin(θ) − Ziel = 0
